# 1C Telegram бот — Инспектор документов

Telegram-бот на 1С для распознавания и проверки документов (счета, накладные).  
Работает через **Long Polling**, использует **OCR Space** для распознавания текста.

## Возможности

- Получение фото документов из Telegram  
- Распознавание текста (номер, сумма, контрагент)  
- Проверка в 1С: контрагент, дубли документов  
- Диалог подтверждения результата распознавания  
- Регламентное задание (Long Polling)

## Требования к инфраструктуре

- Платформа 1С:Предприятие 8.3 (управляемые формы)  
- Серверная или файловая версия (файловая — регламентное задание через таймер формы)  
- Доступ в интернет (Telegram API, OCR Space)

## Настройка

### 1. Создать Telegram-бота
- Найти в Telegram `@BotFather`  
- Создать бота командой `/newbot`  
- Получить токен  

### 2. Получить API-ключ OCR Space
- Зарегистрироваться на `ocr.space`  
- Бесплатный ключ (helloworld) — для теста  

### 3. Доработать конфигурацию 1С

Создайте в своей конфигурации:

**Константы:**

| Имя | Тип |
|-----|-----|
| `tg_token` | Строка (токен Telegram) |
| `ПоследнийID` | Число (смещение сообщений) |

**Регистр сведений:** `СостояниеБота`  
- Периодичность: непериодический  
- Измерение: `ЧатID` (Строка, 50)  
- Ресурсы:  
  - `Состояние` (Строка, 30) – «ОжидаетФото» / «ОжидаетПодтверждения»  
  - `Номер` (Строка, 50)  
  - `Сумма` (Строка, 30)  
  - `Контрагент` (Строка, 150)  

**Регламентное задание:**  
- Имя: `ТгБот`  
- Период: 5–10 секунд  
- Вызов: `ТгЗапрос` (экспортная процедура в обработке)

### 4. Загрузить обработку `.epf` в 1С

- Открыть обработку  
- Указать токен Telegram-бота  
- Включить регламентное задание  

## Принцип работы

1. Пользователь отправляет фото документа боту  
2. 1С через Long Polling получает фото  
3. Сохраняет во временное хранилище  
4. Отправляет в OCR Space  
5. Парсит текст, вытаскивает номер, сумму, контрагента  
6. Запрашивает подтверждение у пользователя  
7. Если «Да» — проверяет в 1С (контрагент, дубли)  
8. Отправляет результат в Telegram  

## Структура обработки

- `ТгЗапрос()` – опрос Telegram, обработка входящих сообщений  
- `СохранитьФотоИЗТг()` – скачивание файла  
- `РаспознованиеOCR()` – вызов OCR Space  
- `ПарсимДанные()` – извлечение номера, суммы, контрагента  
- `ПроверкаВБазеДанных()` – запрос подтверждения  
- `ПользовательОтветилДа()` / `ПользовательОтветилНет()` – проверка в 1С
